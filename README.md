1. Download manhuntHUD by opiumdealer: https://github.com/nastyyaboi/manhuntHUD
2. Download Magic.TXD: https://www.gtagarage.com/mods/show.php?id=27862
3. Extract the PNGs to any folder
4. In manhuntHUD (1.0) files, go to scripts/ManhuntHud.SA/, and open MHud.txd thru Magic.TXD
5. Replace the original mancrouch, manjog, manjunmp, manrun, and manstand by clicking Edit -> Replace or Ctrl+R and selecting Liam PNGs
6. Save MHud.txd
